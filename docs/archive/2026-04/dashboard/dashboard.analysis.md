## ❌ 미구현 항목 상세

### 1. 커스텀 날짜 선택 (Design 2.2.4)
사용자가 임의 기간을 선택해 조회하는 date picker 미구현.

### 2. 활성 필터 표시 텍스트 (Design 2.2.4)
"조회기간: 2026-04-06 ~ 2026-04-13 (7일)" 형태의 텍스트 표시 없음.

### 3. 페이지네이션 (Design 2.2.3)
90일 선택 시 90행이 한번에 표시됨. 20행/페이지 구현 필요.

### 4. 로딩 스피너 (Design 2.1 컴포넌트 트리)
데이터 fetch 중 사용자 피드백 없음.

### 5. 캐싱 전략 (Design 7.2)
5분 TTL 메모리 캐시 미구현으로 불필요한 API 호출 발생.

### 6. 재시도 로직 (Design 7.3)
일시적 네트워크 오류 시 바로 폴백, 3회 재시도 없음.

### 7. KPI 아이콘 (Design 2.2.1)
`<span class="kpi-icon">📊</span>` 요소 없음.
